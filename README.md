nrbib-memory-book
Современный лендинг для Книги Памяти Неклиновского района

Описание проекта
Этот проект представляет собой цифровую книгу памяти Неклиновского района, созданную с использованием современных веб-технологий. Лендинг включает в себя адаптивный дизайн и анимации, что делает его удобным для использования на различных устройствах.

Структура проекта
